# Calculadora CTS 2026 Perú - Ley N.º 32322

Esta es una aplicación profesional desarrollada con **Python** y **Streamlit** para ayudar a los trabajadores peruanos a calcular su Compensación por Tiempo de Servicios (CTS) de forma precisa, considerando la normativa vigente de 2026.

 **Link de la aplicación:** (https://calculadora-cts.streamlit.app/)

##  Características
*   Cálculo Automático: Detecta si el periodo corresponde a Mayo o Noviembre según la fecha de ingreso.
*   Diseño Midnight: Interfaz moderna y optimizada para una mejor experiencia de usuario.
*   Base Legal Actualizada: Incluye la lógica de la **Ley N.º 32322 para el retiro del 100%.
*   SEO Optimized:** Configurada para ser indexada por Google Search Console.

##  Tecnologías Utilizadas
*   Lenguaje:** Python 3.13
*   Framework:** Streamlit
*   Despliegue:** Streamlit Cloud
  
##  Cómo usar
1. Ingresa tu fecha de inicio laboral.
2. Coloca tu sueldo bruto actual.
3. Indica si percibes Asignación Familiar.
4. Haz clic en "CALCULAR" para obtener el desglose detallado de tu beneficio social.
